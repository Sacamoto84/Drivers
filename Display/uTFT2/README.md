Библиотека TFT создана для экранов SSD1306(I2C, SPI), ILI9225, ST7789, ST7735, ST7735S, USB эмуляция, SPRITE_RAM
Работа через буффер фрейма, поддерживается вывод в 1, 4, 8, 16 бит цвете
